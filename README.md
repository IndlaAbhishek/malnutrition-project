# malnutrition-risk-prediction-app

# ML-Based Classifier for Identifying Malnutrition Risk from Basic Health Metrics of Tribal Children

## Project Overview

Malnutrition is a major public health issue affecting millions of children worldwide, especially in tribal and rural areas. Early identification of malnutrition risk can help healthcare workers provide timely intervention and improve child health outcomes.

This project develops a **Machine Learning-based classifier** to identify malnutrition risk levels in children using basic health metrics such as age, weight, height, and other socio-economic indicators.

The system predicts whether a child is at risk of **underweight, stunting, or wasting**, helping health workers make faster and more informed decisions.

* Machine Learning (Scikit-learn)
* Flask API
* Android Application (Frontend)

## Project Structure

```
Backend/
Frontend/
RenderLink.txt
```

* **Backend** – Contains the machine learning model and API used for prediction
* **Frontend** – Android application interface for user input and displaying prediction results
* **RenderLink.txt** – Contains the deployed backend API link

---

## How the System Works

1. User enters child health data in the mobile application.
2. The Android application sends the data to the backend prediction API.
3. The ML model processes the data.
4. The system returns the predicted malnutrition risk level.
5. Results are displayed in the application.

---

## Dataset

The model is trained using data derived from the **DHS Kids Recode dataset**, which includes multiple health and socio-economic indicators related to child nutrition.

---

## Purpose of the Project

The goal of this project is to provide a **simple and accessible digital tool** that can help identify malnutrition risk in children at an early stage, enabling timely healthcare interventions.
